Элементы меню как объекты

В данном задании вам необходимо разобраться с фабричным созданием однотипны объектов. Вы должны составить структуру проекта, таким образом, что каждый элемент меню является целостным объектом и может выполнить свои методы:
  • MenuItem
  • MenuItemExecutor

MenuItem

Должен быть классом с полями в которых будет храниться порядковый начинаюящийся с единицы и заголовок. Данный метод олжен содержать метод print который распечатывает Вывод в консоли должен быть следующим:

Please input required menu item:
1 - Create
2 - Search
3 - Delete
 Так же должны помимо создания файлов реализовать бизнеслогику.
