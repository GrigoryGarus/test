Создаем меню и работаем с System.in

При разработке нам необходимо уметь работать с пользовательским вводом. В Java для работы с терминалльным вводом есть объект System.in. Данный объект может принимать входной поток и считывать данные. Более подробнее о потоках вы должны почитать в книге Философия Java | Эккель Брюс главу 18 Система ввода вывода Java.

В данном задании вы должны реализовать возможность ввода элемента меню с проверкой на тип вводимых данных, если в консоли мы будем вводить не строковое значение, приложение должно выдавать сообщение о неправильном вводе, а затем снова отображать меню.
